# Simple Folder Compare

## Short description

Compare two folders and spot changed files.

## Full description

Simple Folder Compare is a Windows utility designed around a simple, local-first workflow. The initial app gives users a clear starting surface for select left and right folders, compare contents, and review missing, changed, and matching files.

## Feature bullets

- Side-by-side folder setup
- Missing and changed file summary
- Comparison notes
- Local-only review
